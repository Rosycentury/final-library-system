一个基于Spring Boot开发的现代化Web应用，实现了图书的检索、借阅、归还与后台管理功能。

✨ 功能特性
用户管理：读者注册、登录与个人中心。

图书管理：图书信息的增删改查、分类与库存管理。

借阅管理：完整的借书、还书流程，支持逾期提醒。

数据统计：可视化展示借阅排行榜、图书流通数据。
